掼蛋计分器

DEPLOYMENT GUIDE · 部署指南

🌟 方法一：GitHub Pages 免费永久推荐

1
注册 GitHub 账号

访问 github.com 注册免费账号（已有账号跳过此步）

2
新建仓库

点击右上角 + → New repository
仓库名填：guandan（或任意名称）
勾选 Public，点击 Create repository

3
上传计分器文件

点击 uploading an existing file
将下载的 guandan_scoreboard.html 重命名为 index.html
拖入上传框，点击 Commit changes

4
开启 GitHub Pages

进入仓库 → Settings → 左侧 Pages
Source 选择 Deploy from a branch
Branch 选 main，文件夹选 / (root)，点 Save

5
获得网址（约1分钟后生效）
https://你的用户名.github.io/guandan/
全球可访问 · 微信可直接打开 · 永久免费
✅ 优点：完全免费、不限流量、全球 CDN、链接固定不变、微信内直接可用

⚡ 方法二：Netlify 拖拽部署 最简单

1
访问 Netlify Drop

直接打开：app.netlify.com/drop（无需注册）

2
拖拽文件上传

将 index.html（即重命名后的计分器）直接拖入网页中央区域

3
立即获得网址

10秒内自动生成链接，可分享给任何人

https://随机名字.netlify.app
10秒部署 · 注册后可自定义域名
⚠️ 注意：未注册账号的临时链接有效期约 24 小时，注册后永久有效

🔷 方法三：Vercel 免费

1
访问并注册

vercel.com → 用 GitHub 账号登录

2
导入 GitHub 仓库

选择之前创建的 guandan 仓库 → 一键部署

https://guandan.vercel.app
与 GitHub 联动，推送代码自动更新

📱 部署后如何分享

✓
微信分享

直接发链接到微信群，朋友点开即可在微信内置浏览器使用

✓
生成二维码

将链接粘贴到 qr.io 或微信扫一扫，生成二维码打印出来放桌上

✓
添加到手机桌面

Safari/Chrome 打开链接 → 分享 → 添加到主屏幕，像App一样使用
